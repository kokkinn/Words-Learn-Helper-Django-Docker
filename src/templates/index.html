Words Learn Helper
Hi, welcome to the application which provides an opportunity to study foreign language faster. It has three logical sections:

1. Pairs list, Pair creation - you can make a pair of two words which is a main unit to be studied, intended to treat it as a word and it's translation.

2. Groups list, Group creation - a way to join multiple pairs by some category.

3. Tests, Results - check a knowledge of pairs you've created. Test can be started if at least one pair is present.
